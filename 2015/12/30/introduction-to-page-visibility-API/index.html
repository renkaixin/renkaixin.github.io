Javascript判断当前页面/标签是否正在被用户浏览

W3C规范规定了PageVisibility API用以帮助人们判断当前页面/标签是否正在被用户浏览（即页面是否可见），当页面隐藏和现实间切换时，触发visibilitychange事件，通过监听这一事件，可以决策当前页面的行为。
